# Цепочки связанных выражений

Эта возможность позволяет писать сложные связанные выражения типа



```csharp
if (1 < a && a < 5) {}

if (0 < a && a < b && b < c && c < d && d < 255) {
    // do something
}
```

в более естественном виде

```csharp
if (1 < a < 5) {}

if (0 < a < b < c < d < 255) {
    // что-то делаем
}
```
